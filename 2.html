Learning HTML - The Basics of HTML

HTML (Hypertext Markup Language) is the content on a webpage and consists of the following:

  • An opening tag
  • A closing tag
  • Some content

These three components make up an HTML element.

Learning CSS - The Basics of CSS

CSS (Cascading Style Sheets) is the code that styles the content on the page and CSS consists of the following:

  • A selector
  • A property
  • A property value
  • A declaration

These four components make up a CSS ruleset.
